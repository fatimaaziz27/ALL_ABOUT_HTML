#  ALL_ABOUT_HTML

A collection of HTML programs covering the fundamentals of web development, semantic elements, forms, tables, multimedia, and practice exercises. This repository was created to strengthen HTML skills through hands-on coding and real-world examples.

---

## Topics Covered

### HTML Basics

- HTML Document Structure
- Headings & Paragraphs
- Text Formatting
- Comments
- Attributes

### HTML Elements

- Links
- Images
- Lists
- Tables
- Forms
- Buttons
- Iframes

### Semantic HTML

- Header
- Navigation
- Section
- Article
- Aside
- Footer

### Advanced HTML

- Semantic Elements
- Favicons
- Responsive Images
- Accessibility Basics
